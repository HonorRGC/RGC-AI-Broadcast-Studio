# RGC AI Broadcast Studio Roadmap

## Version 0.1 - Foundation
- Connect to iRacing telemetry
- Read running order
- Detect passes
- Push project to GitHub

## Version 0.2 - Race Brain
- Add DriverManager
- Track driver positions
- Track passes made and lost
- Add StoryEngine
- Add event importance ratings

## Version 0.3 - Producer
- Filter low-priority events
- Decide what is worth broadcasting
- Prevent nonstop chatter
- Prioritize lead changes and top-10 battles

## Version 0.4 - Commentary
- Generate broadcaster-style text
- Support two commentators
- Add play-by-play announcer
- Add color commentator

## Version 0.5 - Voice
- Connect ElevenLabs
- Add voice selection
- Add speech queue
- Send audio to OBS

## Version 0.6 - Broadcast Director
- Detect best battle on track
- Suggest camera focus
- Detect replays and big moments
- Add automatic camera switching later

## Version 0.7 - OBS Integration
- Display event overlays
- Show current battle
- Show driver storylines
- Add live broadcast graphics

## Version 1.0 - Public Release
- Windows installer
- User-friendly interface
- League settings
- Voice settings
- One-click start broadcast
